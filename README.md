# P-corridas2019

Instruções para 
- compilar:

```
make
```

- correr:
```
./run
```
É preciso ficheiros `pilotos.txt` e `carros.txt` para que o programa corra, têm se exemplos na pasta `exemples`, copiar para onde o `run` estiver.
